## 🚀 Key Features

*   **Elite Hero Interface:** Full-width, edge-to-edge design with interactive UI motion and professional phone mockups.
*   **Goal-Based Savings:** Intelligent dashboard allowing users to set, track, and visualize financial milestones (e.g., "Buying a House").
*   **Live Transaction Engine:** Functional Deposit/Withdrawal logic with real-time balance updates and historical logging.
*   **Institutional Security:** Dedicated Security Center showcasing 256-bit encryption and Biometric Guard protocols.
*   **Ultra-Responsive Layout:** Zero-gap grid system optimized for Desktop (4K), Tablet, and Mobile devices.
*   **Success Orchestration:** Smooth transition from enrollment (Register) to the dashboard using Framer Motion and custom loaders.

---

## 🛠️ Technical Stack

- **Frontend:** React.js (Vite)
- **Styling:** Modular External CSS (BEM naming convention) & Tailwind CSS
- **Animations:** Framer Motion (Subtle UI floating & smooth page transitions)
- **Icons:** Lucide React (Clean, minimalist iconography)
- **Routing:** React Router DOM v6
- **State Management:** React Hooks (useState, useEffect) & LocalStorage for persistent user sessions.

🎨 Design Philosophy
The UI follows Institutional Minimalism. We prioritize BK Royal Blue (#003366) to establish trust, while using Electric Cyan (#00AEEF) to highlight innovation. Every section is designed to be "Content-Heavy," providing users with deep information typical of a Tier-1 banking institution.
